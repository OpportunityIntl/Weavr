---
title: 'Display Classes'
layout: documentation
---

Weavr comes with a few display classes to show/hide elements at different breakpoints. To only show an element at a specific breakpoint, use `.[breakpoint]-only`. For example, to only show an element on mobile devices (below 480px wide): `.mobile-only`.

To hide an element at a specific breakpoint, use `.no-[breakpoint]`. For example, to hide an element on desktop (above 1024px): `.no-desktop`.

Here's the full list of the display classes: `.desktop-only`. `.netbook-only`, `.tablet-only`. `.mobile-only`, `.no-desktop`, `.no-netbook`, `.no-tablet`, `.no-mobile`.
